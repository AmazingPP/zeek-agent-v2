The Zeek Agent has been installed.

You need to configure a Zeek instance for the agent to connect to. To do so, either execute: 

    sudo defaults write org.zeek.zeek-agent \
            zeek.destination <Zeek host>

Or create a configuration file and add Zeek's address there: 

    cd /Library/Application Support/ZeekAgent
    cp zeek-agent.cfg.template zeek-agent.cfg
    vi zeek-agent.cfg

After any changes, restart the Zeek Agent with: 

    sudo launchctl stop org.zeek.zeek-agent
    sudo launchctl start org.zeek.zeek-agent

Check your Zeek's zeek-agent.log for a successful connection from the agent. If you have any questions or feedback, reach out on the #zeek-agent channel on Zeek's Slack.
